This folder contains resources, especially data.

- Pickle files:
   - factkg_train_5k.pickle: contains 5000 samples from FactKG dataset for training.
   - factkg_test_1k.pickle: contains 1000 samples from FactKG dataset for testing.
   - factkg_dev_300.pickle: contains 300 samples from FactKG dataset for validating.

   - factkg_train_5k_triplets.pickle: contains 5000 samples from FactKG dataset for training, included triplets as tuples.
   - factkg_test_1k_triplets.pickle: contains 1000 samples from FactKG dataset for testing, included triplets as tuples.
   - factkg_dev_300_triplets.pickle: contains 300 samples from FactKG dataset for validating, included triplets as tuples.

- Marisa files:
   - kg_trie.marisa: contains trie from the KG.
