This folder contains resources, especially data.

- Pickle files:
   - factkg_train_5k.pickle: contains 5000 samples from FactKG dataset for training.
   - factkg_test_1k_pickle: contains 1000 samples from FactKG dataset for testing.
   - factkg_dev_300_pickle: contains 300 samples from FactKG dataset for validating.

- Marisa files:
   - kg_trie.marisa: contains trie from the KG.
